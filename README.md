## Setup

Clone the repository

```bash
  git clone https://github.com/Tofazzal-2443/Figma-to-Flutter-Design-1.git
```
Move to the desired folder

```bash
  cd \Figma-to-Flutter-Design-1
```
To run the app, simply write

```bash
  flutter pub get
```
```bash
  flutter run
```
